### What’s this?

I write quite a bit. I have been using [Day One](https://dayoneapp.com) since 2012-10-04 as a journal.  But when it comes to online journaling and/or blogging, I felt like I was always have to compromise on features when you pick any one solution.  I have been meaning to start a blog on a few different topics: technology, roadtrips, donuts, coffee, etc... I use WordPress for my [Tesla Roadtrips](https://kzroadtrip.wordpress.com) but juggled between WordPress and Medium for my topics yet never reached a conclusion.

I really enjoy typing in markdown so it hit me: why not use markdown as the basis for my journaling.  There are some apps I can use to help write entries but there is something nice about busting out a plaintext editor and just hammering away.  With tools like Github Pages and Jekyll, I could push a markdown file and a blog post is ready to go.

Write ups related to code and data will likely remain in their relate repos.  This repo represents writings where my main mode of communication are words.

Enjoy.
